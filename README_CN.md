一个基于Spring Cloud企业级通用构建组件。

## 快速开始
```

<!-- https://mvnrepository.com/artifact/com.wl4g/xcloud-components-common -->
<dependency>
    <groupId>com.wl4g</groupId>
    <artifactId>xcloud-components-common</artifactId>
    <version>${version}</version> <!-- e.g: master -->
</dependency>

```
